This app allows users to interface with their anti-gravity hardware via a local web interface.

## Prerequisites
- Node.js installed on your system.
- Anti-gravity boots or propulsion unit (Model 2.0 or higher).

## How to Run Locally
1. Clone the repository:
   `git clone https://github.com/your-username/anti-gravity-app.git`
2. Navigate to the folder:
   `cd anti-gravity-app`
3. Start the server:
   `npx serve`

## Usage
Once the server is running, open your browser to `http://localhost:3000`. Ensure your hardware is calibrated before toggling the 'Lift' switch.
